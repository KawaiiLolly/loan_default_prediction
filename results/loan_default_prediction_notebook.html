Loan Status Prediction¶

This notebook loads the loan dataset, cleans and preprocesses it, trains multiple classification models, tunes them with GridSearchCV, and exports the best model along with the encoders and scaler for later use in a Streamlit web app.

Model Training with GridSearchCV¶

We train five different models and use GridSearchCV to find the best hyperparameters for each one. cv=5 performs 5-fold cross validation.

In [36]:
results = []
best_estimators = {}

for name, mp in models_and_params.items():
    print(f"Training {name} ...")

    grid = GridSearchCV(
        estimator=mp["model"],
        param_grid=mp["params"],
        cv=5,
        scoring="accuracy",
        n_jobs=-1
    )
    grid.fit(X_train_scaled, y_train)

    best_model = grid.best_estimator_
    best_estimators[name] = best_model

    train_pred = best_model.predict(X_train_scaled)
    test_pred = best_model.predict(X_test_scaled)

    train_acc = accuracy_score(y_train, train_pred)
    test_acc = accuracy_score(y_test, test_pred)

    results.append({
        "Model": name,
        "Best Params": grid.best_params_,
        "Train Accuracy": train_acc,
        "Test Accuracy": test_acc
    })

    print(f"{name} done. Train Accuracy: {train_acc:.4f}, Test Accuracy: {test_acc:.4f}")
    print()
